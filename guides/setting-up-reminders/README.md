# Setting up Reminders

Once the Reminder App is installed on your workspace, the next thing you'll probably want to do is create a reminder. You do this with either:

* Remind yourself about a message _(Most popular feature)_
* `/remind create` slash command
* Using the "Add Reminder" message action button from the message input field
